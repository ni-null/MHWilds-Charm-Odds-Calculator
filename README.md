# MHWilds Charm Odds Calculator

A small frontend tool built with React + Vite for calculating charm (amulet) skill probabilities and combination odds used in Monster Hunter-style systems.

## Key features

- Display a list of available amulets and their skill combinations
- Calculate and present probability distributions across different rarities and numbers of skills
- Multi-language support (Traditional Chinese / English)

## Project structure (short)

- `src/`: application source code
- `src/pages/`: main pages and components
- `src/data/`: game data (for example amulets, skill groups, and base probabilities)
- `public/`: static assets
